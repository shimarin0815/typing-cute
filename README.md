**60秒**で単語を打ちまくる、**スタイリッシュ×ちょい可愛い**タイピングゲーム。  
シンプル構成（HTML/CSS/JSのみ）で、VS Code だけで開発・公開できます。


---

## ✨ 特長
- **リアルタイム判定**：合ってる/ミスを色で表示
- **スコア & ベスト記録**：ブラウザに自動保存（localStorage）
- **WPM/精度**を自動計算
- **難易度切替**（easy / normal / hard）
- **ガラス風カード×パステル**の“かわいモダン”UI
- **ビルド不要**：ファイルを開くだけで動作
